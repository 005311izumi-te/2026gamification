## 対戦ルール（円形リング方式）

教室を円形に並べ、テスト回数分だけ比較相手をずらす。

```
円の並び順：
① 五位堂 → ② 京田辺 → ③ 八木 → ④ 奈良 → ⑤ 学園前
→ ⑥ 木津南 → ⑦ 桜井 → ⑧ 王寺 → ⑨ 生駒 → ⑩ 登美ヶ丘
→ ⑪ 神宮 → ⑫ 西大寺 → ⑬ 郡山 → ⑭ 高の原 → ⑮ 高田 → （①五位堂へ戻る）

テスト①：1つ先の教室と対戦（五位堂 vs 京田辺、京田辺 vs 八木 …）
テスト②：2つ先の教室と対戦（五位堂 vs 八木、京田辺 vs 奈良 …）
テスト③：3つ先、テスト④：4つ先 … テスト⑦：7つ先
```
