# 🏆 プライムコース 教室対抗 小テストバトル ルール説明

## どんな企画？

KEC志学館 プライムコース、全15教室が参加する**教室対抗の小テストバトル**がスタート！

毎週の小テストで、自分の教室が他の教室と対決する。
勝つも負けるも、あなたの点数にかかってる。
**今年の授業は、全部が「勝負」だ。**

## 勝敗のルール

### 対象教科
- 国語
- 英語
- 算数

### 勝利ラインは「80点以上」

各教科、**80点以上を取った人の割合**が多い方の教室が、その教科を勝ち取る！

> 例）五位堂教室で80点以上が6割 vs 京田辺教室で80点以上が4割
> → この教科は**五位堂の勝ち**！

### 3教科で2勝した方が、その週の勝者

| 教科 | 五位堂 | 京田辺 |
|------|--------|--------|
| 国語 | ✅ 勝ち | |
| 英語 | | ✅ 勝ち |
| 算数 | ✅ 勝ち | |
| **結果** | **🏆 勝利！** | |

全教科で気が抜けない、スリルのある3本勝負だ。

## どうすれば勝てるの？

### ズバリ、80点以上を取ること

小テストは、宿題の内容から出題される。
**宿題をしっかりやってくれば、80点以上は十分に狙えるレベル**だ。

> つまり——
> 宿題をやる → 小テストで80点以上 → 教室が勝利に近づく
>
> あなたの努力が、そのまま教室の力になる。

### 1人でも多く、80点を超えることが大事

「どうせ自分が頑張っても…」なんて思わなくていい。
**1人の80点が、教室全体の勝率を上げる。**
強い教室を作るのは、全員の積み重ねだ。

- 「次は絶対リベンジする！」

毎週その瞬間を、楽しみにしてて。
